My Web Application

This web application is the final project for GEOG 585L and visualizes tree census data in New York City. Users can explore tree species, tree health, and spatial distribution interactively using Leaflet and D3.js. The goal is to provide an informative, accessible, and visually engaging way to explore urban forestry data.
